Design a premium enterprise SaaS website for a platform called "Mishti AI".

This is NOT a chatbot product.
This is a "Governed Intelligence Platform" — an AI infrastructure system where every action is validated, traceable, and controlled before execution.

The design must communicate:

* system-level thinking
* enterprise-grade control
* governance-first architecture
* trust, not hype

---

VISUAL STYLE

* Dark theme (primary)
* Background: deep black/navy (#0a0a0f, #0f0f1a)
* Accent colors:

  * Electric Blue (#00d4ff)
  * Neon Teal (#14f195)
  * Subtle Purple (#a855f7)
* Minimal gradients (controlled)
* Glassmorphism (light usage)
* Soft glow effects (not flashy)

Typography:

* Inter font
* Strong hierarchy
* Bold headings (large)
* Monospace font for system indicators

---

DESIGN PRINCIPLES

* This should feel like infrastructure, not a landing page
* Avoid marketing fluff
* Avoid cartoon AI visuals
* Prefer diagrams, flows, and system visuals
* Every section should feel structured and real
* Design should resemble:

  * Stripe docs
  * AWS architecture pages
  * Palantir dashboards
